# TelePsiq_IA — Repositório de Materiais Educacionais

Site estático em HTML para publicação no GitHub Pages.

## Como publicar
1. Criar um repositório no GitHub.
2. Enviar todos os arquivos e pastas deste pacote para a raiz do repositório.
3. Em Settings > Pages, escolher Deploy from a branch.
4. Selecionar Branch: main e Folder: /root.
5. Aguardar a publicação do GitHub Pages.
